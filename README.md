## Connect6

### 설치법(python 3.7.4 기준)

1. git clone
2. python -m venv venv // 가상환경 구성
3. (Window의 경우) >> ./venv/scripts/activate.ps1 
4. (venv 환경에서) pip install -r requirements.txt 
5. python main.py

### 주의사항

윈도우에서만 동작하며, 빌드한 파일의 파일의 경우, 경로 상에 한글이 존재하지 않아야 실행 가능
